天道酬勤？噗...这世界本不公平

发表于 2017-02-14   |     |  

和朋友聊天，她说她总学不会化妆 =。 =
”作为一个女生竟然不会化妆(无情嘲讽)“
”站着说话不腰疼，这东西也要天赋的好吗“
”我觉得努力程度不够，还谈不上说天赋够不够(认真脸)“
”你像梵高一样的努力练习就能成为大师吗“
”……“

”大部分人当失败的时候，只能觉得是自己努力不够吧“

还记得读初中校门口那块石头上刻着：天道酬勤。
诚然，读书是你努力就能有收获的事情。一分耕耘，一份收获。
回头想想，这也是最公平的一次竞争了吧。

成年后，渴望成功。
为完成业务，女人磨断几双高跟鞋，男人喝到身体垮掉。
这些经历甚至能成为日后炫耀的资本，勤奋和努力已深深的印在身上。

“勤奋和其他条件相比，可能只是不起眼的一个”
人们觉得刚开始的苦难和磨难是值得的，但我觉得没有任何的益处，苦难不会带来成功。
苦难不值得追求，磨炼意志力是因为苦难无法躲开。
成功需要运气，天赋，背景，人脉等等，勤奋可能只是不起眼的一个。

但，这并不意味着，你就放弃努力，毕竟那是你唯一的手段。
所以，我们只有努力，我们别无他法。
而且我相信，条件比你好的人更懂得努力的重要性。

世界本不公平，但不公平不是不努力的理由
